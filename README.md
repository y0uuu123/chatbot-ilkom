# chatbot-ilkom
chatbot pembelajaran ilmu komputer berbahasa Indonesia yang dibangun menggunakan NLTK, TF-IDF, dan Cosine Similarity. Sistem ini memproses pertanyaan pengguna, mencari dokumen paling relevan di basis pengetahuan, lalu menampilkan jawaban dengan cepat dan akurat.

# Dataset
- Nama: computer-science-synthetic-dataset
- Jumlah akhir: 20.494 pasangan QA
- Bahasa: Indonesia
- Konten: algoritma, struktur data, pemrograman, komputasi, software engineering
# Dataset melalui proses:
- pembersihan data
- penghapusan duplikat
- normalisasi teks
- pembentukan vektor TF-IDF

# Teknologi yang Digunakan
- Python
- NLTK
- Sastrawi
- Scikit-learn (TF-IDF Vectorizer)
- Flask (Web UI)
- NumPy & Pandas
